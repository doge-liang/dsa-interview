# 学习路线

本文档提供了系统化的数据结构与算法学习路线，帮助你从基础到高级逐步掌握面试所需的算法知识。

## 第一阶段：基础数据结构（1-2 周）

### 目标
- 掌握基本数据结构的操作和时间复杂度
- 理解数组、链表、栈、队列的特性

### 学习内容

#### 1. 数组（Array）
- [ ] 两指针技巧（Two Pointers）
- [ ] 滑动窗口（Sliding Window）
- [ ] 哈希表应用（Hash Table）
- [ ] 前缀和（Prefix Sum）

**推荐题目**：
- LeetCode 1: Two Sum
- LeetCode 15: 3Sum
- LeetCode 167: Two Sum II
- LeetCode 209: Minimum Size Subarray Sum

#### 2. 链表（Linked List）
- [ ] 链表基本操作（增删改查）
- [ ] 快慢指针（Fast/Slow Pointers）
- [ ] 反转链表
- [ ] 环形链表检测

**推荐题目**：
- LeetCode 206: Reverse Linked List
- LeetCode 141: Linked List Cycle
- LeetCode 21: Merge Two Sorted Lists
- LeetCode 19: Remove Nth Node From End

#### 3. 栈和队列（Stack & Queue）
- [ ] 栈的应用（括号匹配、表达式求值）
- [ ] 队列的基本操作
- [ ] 单调栈（Monotonic Stack）
- [ ] 优先队列（Priority Queue）

**推荐题目**：
- LeetCode 20: Valid Parentheses
- LeetCode 739: Daily Temperatures
- LeetCode 239: Sliding Window Maximum

## 面试冲刺（1-2 周）

### 目标
- 综合运用所学知识
- 提高解题速度和准确率

### 建议做法

1. **刷题计划**
   - 每天做 3-5 道 LeetCode 中等难度题目
   - 周末做 1-2 道困难题目
   - 限时训练（每道题 20-30 分钟）

2. **分类练习**
   - 专项突破薄弱环节
   - 重做错题和难题

3. **模拟面试**
   - 使用 LeetCode 面试模式
   - 练习口述思路

4. **总结归纳**
   - 记录常见的解题模式和技巧
   - 整理错题本
